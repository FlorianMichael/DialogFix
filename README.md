# DialogFix

Selects the first input field when opening a dialog.

## Downloads

Modrinth - https://modrinth.com/mod/dialogfix

Dev builds - https://build.florianreuth.de/job/DialogFix

## Use in Gradle

To use DialogFix with Gradle you can
use [my own repository](https://maven.florianreuth.de/#/releases/de/florianreuth/dialogfix).  
You will find instructions on how to add it into your build script there.
